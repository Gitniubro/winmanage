# Windows 管理小工具

14 个独立的批处理文件，每个文件均可双击独立运行，无需主菜单入口。

## 文件说明

| 编号 | 文件名 | 功能描述 |
| ---- | ------ | -------- |
| 01 | `01_右键菜单设置.bat` | 切换 Windows 10/11 右键菜单样式、添加/删除超级菜单、添加/删除文件 Hash 校验右键菜单 |
| 02 | `02_桌面设置.bat` | 隐藏/显示桌面图标小箭头、Windows 聚焦、桌面壁纸管理、Bing 每日壁纸、版本水印 |
| 03 | `03_任务栏设置.bat` | 一键净化任务栏、小组件开关/卸载/安装、任务视图、搜索、自动隐藏、时间秒数显示 |
| 04 | `04_资源管理器设置.bat` | 默认打开路径、扩展名开关、单击/双击打开、复选框、系统隐藏文件、U 盘开关、导航栏开关、缓存清理 |
| 05 | `05_Windows更新设置.bat` | 暂停 Windows 更新至 2999 年、恢复更新 |
| 06 | `06_UAC设置.bat` | 从不通知、恢复默认、彻底关闭/启用 UAC、打开 UAC 手动设置界面 |
| 07 | `07_系统设置集中管理.bat` | 打开上帝模式控制面板（所有系统设置集中一处） |
| 08 | `08_WIFI密码.bat` | 显示本机所有已连接过的 WiFi 名称及密码 |
| 09 | `09_电源管理.bat` | 定时关机/重启/休眠、禁用自动睡眠、电源选项、启用/禁用休眠 |
| 10 | `10_应用管理.bat` | 一键卸载预装应用（winget）、程序和功能、OneDrive 安装/卸载、微软拼音输入法设置 |
| 11 | `11_编辑Hosts.bat` | 以管理员权限打开 hosts 文件 |
| 12 | `12_网络管理.bat` | 网络信息、DNS 缓存、MAC 地址、Ping、路由追踪、外网 IP、端口占用、远程桌面、防火墙、代理、端口转发 |
| 13 | `13_设备管理.bat` | 禁用/启用照相机、蓝牙 |
| 14 | `14_Hash计算.bat` | 计算文件的 MD2/MD4/MD5/SHA1/SHA256/SHA384/SHA512 哈希值 |

## 使用说明

1. **双击任意 bat 文件即可运行**  
   每个脚本开头均已内置管理员权限自提（`net session` + `powershell Start-Process -Verb RunAs`），无需手动以管理员身份运行。

2. **主题配色**  
   所有脚本共用 `%APPDATA%\Windowsbat\config.cmd` 保存的主题颜色，可在任意脚本中设置后全局生效。

3. **系统兼容性**  
   - 请在 Windows 11/10 中使用
   - 建议使用简体中文系统，以确保字符显示正常
